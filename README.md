# Plataforma Académica — Universidad Continental

Sistema de gestión académica para el control de estudiantes y cursos desarrollado como parte del proyecto PA3.

## 🛠 Tecnologías Utilizadas
* **Backend:** ASP.NET Core (C#)
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Base de Datos:** SQL Server
* **Entorno:** .NET 8.0

## 🚀 Cómo ejecutar el proyecto
1. Clonar el repositorio: `git clone <tu-url-de-repo>`
2. Navegar a la carpeta del backend:
   ```bash
   cd backend\EstudiantesAPI

   Ejecutar la aplicación:
   dotnet run

   Abrir el archivo index.html en tu navegador.
